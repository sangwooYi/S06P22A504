# 공통코드

## 목적

> 우유마켓 서비스를 구현하면서 서비스 내에서 공통적으로 사용하는 수많은 코드 데이터들을 한데 모아서 관리하기 위한 목적으로 설계하였습니다. 중고거래 서비스 특성상, `거래 구분, 상품 카테고리, 상품거래 상태`으로 구분하였습니다.
